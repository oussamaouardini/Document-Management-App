Frequently Asked Questions

Where are files actually stored?

When you upload a file to your Info Sat platform storing , that file is now stored in Fire base server network, which is provided by Google and comprised of many data centers spread all over the world.

What types of files can I store?

You can store virtually any file type in your account,including Docs, Sheets, Images,and Slides... which means that all your files can be stored and opened within your account.

How do I manage Accessing to the files?

You can set role-based access controls at the Uploads level. You can then restrict and permit individuals or groups to access to the files .

May I Update files Accessing and data ?

You can set only your file description or Accessing in your "my posts" component .
